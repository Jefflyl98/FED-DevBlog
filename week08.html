Week 8 Dev Blog

Fetch API

  • Data Attributes
    • Store information in an HTML element
    • data-[name]="data" -- Add data attribute
  • Getting and Setting Data Attributes
    • document.querySelector() - get and set data attributes of an HTML element
    • .getAttribute(attrbName) - GET
    • .setAttribute(attrbName, value) - SET
  • Asynchronous Javascript and XML (AJAX)
    • Load data without refreshing the page and no interruptions when doing other activities
      • Browser requests information from server
      • Data processed on the server and returned back to browser
      • Browser process and adds the data to the page
      TAKE NOTE *Outdated Method*
  • API
    • Request and process the data and returns it
    • Called using fetch method
    • Methods for calling an api
      • GET - Retrieve data
      • POST - Add data
      • PUT - Update data
      • DELETE - Delete data
    • Data is sent in JSON format to the API. Hence, encoding needed from a Javascript Object, using JSON.stringify()
