Week 7 Dev Blog

DOM Events Local Storage

  • Document Object Model(DOM)
    • DOM Tree - A model of the page when the browser loads it
    • Every element, attribute, text in HTML is represented by its own DOM node
  • Accessing Elements
    • getElementById() - Finds the node using the element's id attribute
    • quertSelector() - Finds the first matching node using a CSS selector
    • getElementsByClassName() - Finds all nodes with the tag name
    • querySelectorAll() - Finds all elements using a CSS selector
  • Event Handling & Validation
    • element.addEventListener('event', functionName - Calls the function, [Boolean])
      • EventListeners can be used to validate a user's form input when the form is submitted
    • UI Events occur when a user interacts with the browser's UI rather than the webpage
    • Keyboard Events occur when a user interacts with the keyboard
    • Mouse Events when a user interacts with a mouse,trackpad or touchscreen
    • Focus Events when an element gains or loses focus
    • Form Events when the user interacts with a form element
  • Local Storage
    • Local and Session storage
      • Session Storage - data lost when tab is closed, open windows/tabs cant access it's data
      • Local Storage - data is not lost when tab closed, allows open windows/tabs to access the data
    • Storing of key-value pairs in the browser
    • Maximum data storage: 5 MB per domain
  • Javascript Object Notation [JSON]
    • Looks like Javascript Object
    • Data stored in a key-value pair
    • Can be parsed into Javascript Object using JSON.parse()
